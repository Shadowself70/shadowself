# Harmonic Resonance Site Worker

This Worker attaches the custom domain `harmres.shadowself.ca` and proxies the published Harmonic Resonance game page from `https://shadowself.ca/games/harmres.html`.

## Files

- `wrangler.jsonc` - Worker config and custom domain route
- `src/worker.mjs` - Serves the Harmonic Resonance page at the domain root
