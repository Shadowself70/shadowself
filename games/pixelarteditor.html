How to Use PIXELEDITOR

🎨 Drawing & Tools

  • Pencil (B): Draw pixels. Click and drag.
  • Eraser (E): Erase pixels (transparent).
  • Bucket (F): Fill connected areas with color.
  • Eyedropper (I): Pick a color from the canvas.
  • Move (M): Click and drag to shift the sprite within the frame.
  • Line (L): Drag to draw straight lines.
  • Rect (R): Drag to draw a rectangle outline.
  • Circle (C): Drag from center to draw a circle.

🎞️ Animation & Layers

  • Background Layer: Select the "BACKGROUND" card in the Frames list to edit the static background layer. This layer sits behind all animation frames. Use the '🗑️' button to clear it.
  • Frames: Click a frame to edit it. Drag and drop frames to reorder them.
  • Onion Skin: Shows a ghostly transparent version of the previous frame to help you animate movement.
  • Preview: Use the dropdown to zoom in (2x, 4x, 8x).

💾 Exporting

  • Sprite Sheet: Exports all animation frames side-by-side as a PNG. Useful for game engines. Does NOT include the background layer.
  • GIF: Exports a playable animation. Includes the background layer.
  • Export Scale: Use the input in the top bar to resize your export (e.g., 4x makes a 32px sprite into 128px).
  • Save Project: Saves a .json file to edit later.

⌨️ Shortcuts

Ctrl + Z : Undo
Ctrl + Y : Redo
Space : Play/Pause
B : Pencil
E : Eraser
L : Line
